How to set github 'personal access token'

This post is to solve ‘Support for password authentication was removed. Please use a personal access token instead’ when using hexo deploy

Find the settings

  • Settings -> Developer settings -> Personal access tokens -> [Generate new token]

Set the token profile

  • Give it a note, expiration time (for static html deploy, it would be NO EXPIRATION), and the scopes (just [repo] for static html)

  • And you will get a token like:

    ghp_*****************************

  • Copy it

Open Keychain Access.app

  • Paste the token copied before into the password field

DONE!
